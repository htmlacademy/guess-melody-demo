# Список внешних зависимостей

Ниже представлен полный список пакетов, которые используются для разработки демонстрационного/личного проекта. Обратите внимание, все пакеты должны быть установлены определённой версии. Версия пакета вводится после символа `@`. Например: 

```
npm i <package>@<version> -E
npm i react@16.14 -E
```
